## Poetry add packages

~~~ bash
poetry add --group dev ruff pylint isort pre-commit
~~~

## Run commands

~~~ bash
poetry run isort .
~~~

~~~ bash
poetry run ruff check .
poetry run ruff check . --fix --unsafe-fixes
~~~

~~~ bash
poetry run pylint src/*
~~~

## RUN app

~~~ bash
poetry run python3 src/app.py
~~~
